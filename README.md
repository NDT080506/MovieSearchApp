# Movie Search App

Ứng dụng tìm kiếm phim được xây dựng bằng HTML, CSS và JavaScript.

## Chức năng

- Tìm kiếm phim
- Lọc theo thể loại
- Sắp xếp theo đánh giá
- Phân trang

## Công nghệ

- HTML
- CSS
- JavaScript

## Cách chạy

1. Clone repository:

```bash
git clone https://github.com/NDT080506/MovieSearchApp.git
```

2. Mở file `index.html` bằng trình duyệt.
